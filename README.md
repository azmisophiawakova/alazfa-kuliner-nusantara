## Deskripsi Alazfa Kuliner Nusantara
Aplikasi yang akan dikembangkan adalah Alazfa Kuliner Nusantara, yaitu sebuah aplikasi berbasis web yang berfungsi sebagai platform pemesanan makanan dan minuman tradisional dari berbagai daerah di Indonesia. Aplikasi ini dirancang untuk menghubungkan pelanggan, penjual (UMKM kuliner), kurir, dan admin dalam satu sistem yang terintegrasi. Secara umum, aplikasi ini tidak hanya menyediakan layanan pemesanan makanan secara online seperti platform layanan pesan-antar pada umumnya, tetapi juga memiliki nilai tambah berupa fitur edukasi. Fitur tersebut berupa informasi resep, bahan, serta deskripsi dan filosofi dari setiap menu makanan tradisional. Dengan demikian, aplikasi ini tidak hanya berorientasi pada transaksi, tetapi juga berkontribusi dalam pelestarian budaya kuliner nusantara.
